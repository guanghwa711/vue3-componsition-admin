> vue3-composition-admin 是一个管理端模板解决方案，它是基于vue3,ts和element-plus，项目都是以composition api风格编写。

## 简介

项目都是出自于源于花裤衩大佬的 vue-element-admin，之后有了各种版本。vue3 发布之后，性能和体积都在减小，composition api 增加了项目可读性，决定入坑，重写一遍加深练习以便以后使用。
